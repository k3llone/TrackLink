# SD-23 — API: изменить ссылку

```mermaid
sequenceDiagram
    autonumber

    actor Client as External API Client
    participant API as PublicAPIHandler
    participant AA as ApiKeyAuth
    participant RL as RateLimiter
    participant KR as ApiKeysRepository
    participant LS as LinksService
    participant LR as LinksRepository
    participant DB as PostgreSQL
    participant RD as Redis

    Client->>API: PATCH /api/v1/links/{link_id} + API key
    API->>AA: Authenticate(api_key)

    AA->>KR: FindByRawKey(api_key)
    KR->>DB: SELECT api_key by hash/prefix
    DB-->>KR: key row or no rows
    KR-->>AA: key or not found

    alt API key невалиден или отозван
        AA-->>API: Unauthorized
        API-->>Client: 401 Unauthorized
    else API key валиден
        API->>RL: CheckLimit(api_key)
        RL->>RD: INCR rate_limit counter
        RD-->>RL: allowed / blocked
        RL-->>API: allowed / blocked

        alt Rate limit exceeded
            API-->>Client: 429 Too Many Requests
        else Запрос разрешён
            API->>LS: UpdateLink(user_id, link_id, target_url, status)

            LS->>LR: GetByIdAndOwner(link_id, user_id)
            LR->>DB: SELECT link by id and owner_id
            DB-->>LR: link row or no rows
            LR-->>LS: link or not found

            alt Ссылка не найдена или не принадлежит пользователю
                LS-->>API: LinkNotFound
                API-->>Client: 404 Not Found
            else Ссылка найдена
                LS->>LR: UpdateLinkFields(link_id, target_url, status)
                LR->>DB: UPDATE links SET target_url = ..., status = ..., updated_at = now()
                DB-->>LR: OK
                LR-->>LS: LinkUpdated

                LS->>LR: GetByIdAndOwner(link_id, user_id)
                LR->>DB: SELECT updated link
                DB-->>LR: updated link row
                LR-->>LS: updated link

                LS-->>API: LinkUpdatedResult
                API-->>Client: 200 OK + updated link
            end
        end
    end
```
