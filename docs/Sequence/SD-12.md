# SD-12 — Активация / деактивация ссылки

```mermaid
sequenceDiagram
    autonumber

    actor Customer
    participant FE as Frontend
    participant LH as LinksHandler
    participant LS as LinksService
    participant LR as LinksRepository
    participant DB as PostgreSQL

    Customer->>FE: Меняет статус ссылки (active / inactive)
    FE->>LH: PATCH /links/{link_id}/status

    LH->>LH: Валидирует payload
    alt Некорректный payload
        LH-->>FE: 400 Validation Error
        FE-->>Customer: Показать ошибку
    else Данные валидны
        LH->>LS: ChangeStatus(user_id, link_id, new_status)

        LS->>LR: GetByIdAndOwner(link_id, user_id)
        LR->>DB: SELECT link by id and owner_id
        DB-->>LR: link row or no rows
        LR-->>LS: link or not found

        alt Ссылка не найдена или не принадлежит пользователю
            LS-->>LH: LinkNotFound
            LH-->>FE: 404 Not Found
            FE-->>Customer: Показать ошибку
        else Ссылка удалена или заблокирована
            LS-->>LH: StatusChangeNotAllowed
            LH-->>FE: 409 Conflict
            FE-->>Customer: Показать ошибку
        else Смена статуса разрешена
            LS->>LR: UpdateStatus(link_id, new_status)
            LR->>DB: UPDATE links SET status = ..., updated_at = now()
            DB-->>LR: OK
            LR-->>LS: StatusUpdated

            LS->>LR: GetByIdAndOwner(link_id, user_id)
            LR->>DB: SELECT updated link
            DB-->>LR: updated link row
            LR-->>LS: updated link

            LS-->>LH: StatusChanged
            LH-->>FE: 200 OK + updated link
            FE-->>Customer: Показать новый статус ссылки
        end
    end
```
